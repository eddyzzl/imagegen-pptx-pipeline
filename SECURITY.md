# Security

This project is a workflow skill and does not intentionally handle credentials.

## Do Not Commit

- Private templates
- Internal decks
- Personal data
- Business metrics that are not public
- Generated customer deliverables
- API keys, tokens, cookies, or credentials

## Reporting Issues

If you find a security or privacy issue, open a private report with:

- A short description
- Affected file or workflow phase
- Minimal reproduction steps
- Suggested remediation, if known

## Local Data Handling

Pipeline workspaces may contain user inputs, generated images, and PPTX outputs. Keep them outside git and clean them before sharing logs.
